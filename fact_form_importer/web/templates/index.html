Archived runs

What the columns mean
Run
The archived import identifier. Select it to open that run.
Completed
When workbook ingestion and archive creation finished.
Workbook
The source Microsoft Forms export.
Source submissions
Every non-header form row before duplicate suppression.
Authoritative submissions
The latest retained submission for each court, used for validation, review and API planning.
Superseded submissions
Older duplicate submissions retained for audit but excluded from operational processing. Authoritative plus superseded submissions equals the source total.
Ingestion result
The immutable outcome for authoritative submissions. These four figures always add up to the authoritative total.
Unique courts needing review
The live, deduplicated number of courts with known human work remaining. It changes as decisions are recorded.
API plan
The current courts with a confirmed FaCT target, their planned section actions and completed actions. It excludes unmatched courts, but can grow when a reviewer supplies the correct FaCT court slug.
