What this step does

This is not a {{ comparison_summary.total }}-item approval queue. New workbook runs fetch current FaCT sections automatically after publishing the archive. This page shows those prepared comparisons; you review only real changes to populated live sections.

  • No approval is needed when FaCT is empty or the effective result is unchanged.
  • An approval is requested only when submitted values would change a populated live section.
  • High-confidence LLM values are already automatically approved. This page has no model confidence of its own: its separate approval protects populated live FaCT data from an unintended overwrite.
  • Submitted values are merged into live FaCT data. Blank and unmatched live values are preserved, and a later live or proposed change invalidates approval.
